# Soccer_analysis
## About the Project

This project will look at goal information from the top level men's soccer teams in the UK Italy and Spain these are **Premier League**, Series 8, and *La Liga* respectively. We will look at scoring trends of the last five years and compare these between the three leagues.

###### Credits: [Kaggle: Europe, big 5 league last season match scores](https://www.kaggle.com/datasets/sinansaglam/europe-big-5-league-last-5-season-match-scores)
